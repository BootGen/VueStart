## Introduction

An online tool that generates data tables using pure Vue.js. Input some JSON data, chose a template, download the code and use it in any project. Try it online at [vuestart.com](https://vuestart.com)!


### Features

 - Pagination
 - Sortable columns
 - Navigation for nested data structures
 - Easily theme-able

### Why use Vue Start instead of a data table library?

It is today's tendency that programmers use libraries far too often. Every time they do so, they add yet an other dependency, yet an other security risk to their software, making it harder to maintain. Vue Start creates high quality, ready-to-use source code, that is easy to integrate into any project, without adding any dependency. Integrating source code instead of a library gives you freedom over your project. No one can anticipate every future needs. Libraries will have their shortcomming, and so will our code, but while it is difficult to change a library, it is very easy to change our source code.

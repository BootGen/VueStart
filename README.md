## Introduction

Create well coded Vue 3 tables with Bootstrap, Tailwind CSS or vanilla CSS. Try it online at [vuestart.com](https://vuestart.com)!


### Features

 - Pagination
 - Sortable columns
 - Navigation for nested data structures
 - Easily theme-able
